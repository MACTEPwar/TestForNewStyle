# TestForNewStyle

Для создания подключения к БД необходжимо изменить настройки "ConnectionString" в файле appsetting.json в корне проекта

Для создание БД и таблиц необходимо в Консоли диспетчера пакетов прописать "update-database".

Для добавления записей выбрать соответствующий пункт в навигационном меню.

Для работы ajax запросов необходимо в каждом js-файле прописать свое доменное имя
